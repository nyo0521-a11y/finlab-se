# gpt-image-2プロンプト：冒頭挿絵「全額返すか、一部だけ返すか」気持ちの対比（v2）

記事: 住宅ローンの金利上昇、一部繰り上げ返済にした理由と手数料（`content/drafts/mortgage-partial-prepayment.md`）
配置: リード文の直後
保存先: `static/images/illust-mortgage-partial-prepayment-hero.png`

## v2の設計意図（ユーザーフィードバック反映）

- v1（白基調ミニマル・分かれ道と家のアイコン）は「シンプルすぎる」との評価で不採用。**親しみやすくリッチな、gpt-image-2の描画力を活かした場面イラスト**に方針転換
- 分かれ道・家のモチーフは使わない。**「気持ちの対比」**を主役にする：
  - 全額繰り上げ返済のメリット側＝金利上昇の不安・金利負担が軽くなり、頭がすっきりして気持ちが楽になる
  - 気になるポイント側＝繰上げ返済手数料がかかる、団信がなくなる
- どちらかを悪者化しない中立バランス（記事の結論は「手数料で一部を選んだ」だが、イラストでは誘導しない）
- 最新ChatGPTは日本語描画が安定しているため、文字数制限は撤廃（「繰上げ返済手数料」「団信がなくなる」等そのまま使用）。保険として「一字一句正確に」の指示だけ残す

## プロンプト本文（ChatGPTにそのまま入力）

```
日本語の個人投資ブログの記事冒頭に挿入する、読者の共感を誘う温かみのあるイラスト（1376×768px・16:9・横長）を作成してください。
画像内に描く日本語テキストは、指定した文言を一字一句正確にレンダリングしてください。

【テーマ】
住宅ローンを「全額繰り上げ返済してしまうか、一部だけにするか」で心が揺れている40代前半の会社員。全額返してしまえば気持ちが楽になる一方で、手数料や団信のことが少し気になる——そんな「気持ちの対比」を1枚で伝えるイラストです。

【全体のスタイル】
- 温かみのある絵本調・手描き風のデジタルイラスト。柔らかい色鉛筆や水彩のような優しい質感で、光や奥行きも感じられるリッチな描き込み
- 明るく親しみやすい雰囲気。ブログ本文（白背景）に馴染む、明るめの背景色
- 人物は親しみの持てるキャラクター表現（デフォルメ可）。ただし実在の人物を特定できるような写実的な顔立ちにはしない

【構図】
- 中央に40代前半の会社員（ビジネスカジュアル・襟付きシャツ）。ソファやデスクでくつろぎながら、少し首をかしげて考えごとをしている。深刻ではなく「うーん、どっちにしようかな」という穏やかな表情
- 人物の頭上に、ふわふわした大きな思考の吹き出し（もくもくした雲形）を左右に2つ描き、気持ちの対比を表現する：
  - 【左の吹き出し：全額返済したときの気持ち】晴れやかな青空と暖かい日差しの下で、同じ人物が両手を伸ばして深呼吸し、肩の荷が下りてすっきりした表情。頭上から「金利上昇」「金利負担」と書かれた灰色の重そうな雲がすーっと遠くへ流れ去っていく。開放感・身軽さが伝わる明るい色調
  - 【右の吹き出し：ちょっと気になること】同じ人物が少し眉を下げて頬に手を当て、2つのものを見つめている。1つは「繰上げ返済手数料」と書かれた伝票・請求書。もう1つは「団信」と書かれた盾が半透明になって消えかけている様子。悲観的ではなく「ここだけ気になるな」という軽い引っかかりのトーン
- 2つの吹き出しは同じくらいの大きさにして、どちらが正解かを示唆しない
- 全体の余白や小物（観葉植物、マグカップ、ノートパソコンなど）で生活感と親しみやすさを演出してよい

【色】
- 全体は暖色系の調和したパレット（クリーム・ベージュの背景、暖かいオレンジや優しいグリーンの差し色）
- 左の吹き出しは明るく開放的な色調、右の吹き出しはわずかに落ち着いた色調。ただし右側を暗く重くしすぎない

【画像内に描いてよい日本語（これ以外の文字は描かない）】
- 「金利上昇」「金利負担」（左の吹き出しの流れ去る雲に）
- 「繰上げ返済手数料」（右の吹き出しの伝票に）
- 「団信」（右の吹き出しの盾に）

【絶対にやらないでほしいこと】
- 上記以外の文字・数字・ロゴ・銀行名を描かない
- 分かれ道や家のモチーフを主役にしない
- 悲壮感・深刻な表情・真っ暗な色調にしない（あくまで前向きに検討している場面）
- チェックマークや✕マークで良し悪しを示さない
- 実在の人物・有名人を想起させる顔立ちにしない
```

## 生成後の確認

1. 4つのラベル（金利上昇／金利負担／繰上げ返済手数料／団信）が一字一句正確か
2. 左右の吹き出しの「気持ちの対比」が一目で伝わるか（左=すっきり開放感、右=軽い気がかり）
3. 全体が親しみやすい温かいトーンか（深刻・悲壮になっていないか）
4. 崩れ・違和感があれば該当箇所だけ指摘して再生成
