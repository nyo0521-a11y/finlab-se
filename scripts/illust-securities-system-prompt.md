## イラスト①：多要素認証3方式の比較
**ファイル名：** `static/images/illust-mfa-comparison.png`
**サイズ：** 1200×630 px

### ChatGPTに貼り付けるプロンプト

```
横長のインフォグラフィック画像（1200×630px）を作成してください。

テーマ：「多要素認証3方式の比較」
日本語のテキストを正確にレンダリングしてください。

レイアウト：
画面を縦に3分割し、3つの認証方式を左から右へ並べる。

【方式1：SMS認証】（左カラム）
- アイコン：スマホからSMSメッセージが届くイメージ（吹き出しに6桁数字「123456」）
- 見出し：「SMS認証」
- 説明：「スマホにSMSで届く6桁コードを入力」
- 耐性レベル表示：「△ 弱い」（オレンジ色のバッジ）
- 弱点：「SIMスワップ・偽サイト入力」

【方式2：TOTP（認証アプリ）】（中央カラム）
- アイコン：スマホ画面に認証アプリ（時計アイコン）と6桁コードを表示
- 見出し：「TOTP（認証アプリ）」
- 説明：「Google Authenticator等のアプリが30秒ごとに変わるコードを生成」
- 耐性レベル表示：「○ 中程度」（黄色のバッジ）
- 弱点：「リアルタイム中継攻撃に弱い」

【方式3：パスキー（FIDO2）】（右カラム）
- アイコン：スマホと鍵マーク（指紋認証や顔認証イメージ）
- 見出し：「パスキー（FIDO2）」
- 説明：「端末内の秘密鍵で、サイトのURLと連動して認証」
- 耐性レベル表示：「◎ 強い」（緑色のバッジ）
- 強み：「フィッシング原理的に防御」

下部に小さく：「フィッシング耐性の差」

スタイル：
- 背景：白〜淡いグレーグラデーション
- 色調：オレンジ（弱い）→黄色（中程度）→緑（強い）の段階的なカラーコーディング
- フラットデザイン、線画ベース
- 日本語フォントは游ゴシック相当の太字
- 各カラムは薄い枠線で区切る
- カードの上部に方式名、中央にアイコン、下部に説明と耐性バッジを配置

絶対にやらないでほしいこと：
- 暗い背景にしない
- 写真風・3D CGにしない（フラットなインフォグラフィック）
- 英語に変換しない（日本語のまま正確に）
```

## イラスト②：パスキーがフィッシング攻撃に強い理由

**サイズ：** 1200×630 px

### ChatGPTに貼り付けるプロンプト

```
横長のインフォグラフィック画像（1200×630px）を作成してください。

テーマ：「パスキー（FIDO2）がフィッシング攻撃に強い理由」
日本語のテキストを正確にレンダリングしてください。

【上段：本物サイトでの動作（成功）】

- 中央に矢印（緑色・「鍵で認証」）
```

## イラスト③：不正取引検知（FDS）の仕組み
**ファイル名：** `static/images/illust-fraud-detection-system.png`
**サイズ：** 1200×630 px

### ChatGPTに貼り付けるプロンプト

```
横長のインフォグラフィック画像（1200×630px）を作成してください。

テーマ：「不正取引検知（FDS：Fraud Detection System）の仕組み」
日本語のテキストを正確にレンダリングしてください。

レイアウト：左から右へのフロー図（4ステップ）。

【ステップ1：通常パターンの学習】
- アイコン：折れ線グラフ＋AI/データベース風アイコン
- 見出し：「① 通常パターンを学習」
- 説明：「ログイン時間・取引銘柄・出金パターン等を継続的に記録」

→ 右へ矢印

【ステップ2：取引発生】
- アイコン：取引画面（PC・スマホ）
- 見出し：「② 取引・出金が発生」
- 説明：「ユーザーが認証を経て取引を実行」

→ 右へ矢印

【ステップ3：異常検知】
- アイコン：警告マーク（黄色の三角に「!」）
- 見出し：「③ パターンと比較・異常検知」
- 説明：「普段と違う時間・銘柄・出金先を検出」
- 検知例（吹き出し3つ）：
  - 「普段使わない端末からのアクセス」
  - 「普段売買しない銘柄を急に大量取引」
  - 「出金先変更直後の大口出金」

→ 右へ矢印

【ステップ4：一時停止＋本人確認】
- アイコン：一時停止マーク（赤い●に縦バー2本）＋電話・メール
- 見出し：「④ 取引を一時停止・本人確認」
- 説明：「電話やメールで確認、本人なら継続、不正なら被害を防止」

下部にまとめ：「『不審な取引を検知しました』アラートはFDSが守ってくれている証拠」

スタイル：
- 背景：白ベース
- 色調：青系（信頼）をベースに、警告は黄色・停止は赤
- フローを示す太い矢印で4ステップを連結
- 各ステップは薄いブルー背景のカード型
- フラットデザイン、シンプルなアイコン
- 日本語フォントは游ゴシック相当の太字
- カード上部にステップ番号と見出し、中央にアイコン、下部に説明を配置

絶対にやらないでほしいこと：
- 暗い背景にしない
- 写真風・3D CGにしない（フラットなフロー図）
- 英語に変換しない（日本語のまま正確に）
- 攻撃者・ハッカーのイラストを入れない
```
