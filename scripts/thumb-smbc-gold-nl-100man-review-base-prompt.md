# サムネイル ベース画像生成プロンプト

記事: 三井住友カード ゴールド(NL)の100万円修行は本当に得か（金融SEが損益分岐を正直レビュー）
ファイル: `static/images/thumb-smbc-gold-nl-100man-review-base.png`

## 使い方

1. **新しいChatGPT会話**を開始（過去の文脈を引きずらないため）
2. 既存サムネ1〜2枚（`thumb-sbi-securities-honest-review-base.png` 等）を添付し「このテイストで作って」と伝える
3. 以下のプロンプトをコピペして渡す
4. 生成されたベース画像をDownloadsに保存（テキスト合成はローカルで行うのでChatGPTでは文字を入れない）

---

```
日本語の個人投資・クレカ解説ブログ用のサムネイル画像（1376×768px・16:9・横長）を作成してください。

【重要：これは「三井住友カード ゴールド(NL)の"100万円修行"（年間100万円利用で年会費を永年無料にする）を達成した実体験レビュー」記事のサムネイルです。白背景のインフォグラフィックではなく、シネマティックなダークCG調の画像を作成してください。】

【画像全体の雰囲気】
- ダークネイビーから漆黒のグラデーション背景（#050C23 → #0A1A4A）
- 高級感のある金融テックの世界観（BloombergやNikkeiの金融ビジュアルに近い）
- 全体に薄い回路パターン／グリッドラインを敷く（深い青色で控えめに）
- 写真のようなリアル3D CG調
- テキスト・ロゴ・ラベル・カード券面の文字は一切入れないこと
- 後でテキストを左側に重ねるので、画面左45%は暗く保ち、目立つオブジェクトを置かないこと

【ビジュアル構成】
画面右55%のエリアに、以下を配置：
- 主役：高級感のあるゴールドのクレジットカード1枚を、立体的に浮かせて斜めに配置（メタリックな金色に発光、表面に光沢とリフレクション。券面のロゴ・数字・文字は入れない、無地の上質なゴールドカードとして描く）
- カードの背後：緑色に発光する立体的な上昇チャートのライン（つみたて投資・資産成長のイメージ）が奥行きを持って走る
- カードの周囲：少量の金色のコインがやわらかく舞い、上品な光の粒子（緑・金）が漂う
- カードから放たれる放射状のソフトグロー
- アクセント色：エメラルドグリーン #4ADE80、アンバーゴールド #F59E0B

【画面左45%エリア】
- 暗いまま保ち、薄いグリッドラインだけが見える程度
- 目立つオブジェクトを置かない（後でテキストを重ねるため）

【カラーパレット】
- 背景：#050C23 → #0A1A4A
- メイン：#F59E0B（アンバーゴールド＝カードの金色）／#4ADE80（エメラルドグリーン＝チャート）
- 粒子：緑と金の混色
- グリッドライン：深い青、非常に薄く

【サイズと形式】
- 1376×768px のPNG
- アスペクト比 16:9 横長
- テキスト・ロゴは絶対に入れないこと

【絶対にやらないでほしいこと】
- 白背景のインフォグラフィックにしない
- 4分割やテーブル形式のレイアウトにしない
- ✓や✗のチェックマークを入れない
- カード券面に文字・数字・ブランドロゴ・実在の社名を入れない
- 前回記事で主役にした「整然と積み上がるコインの柱」「格子状に並ぶ多数のタイル」を主役にしない（今回の主役はあくまでゴールドカード1枚）
```
