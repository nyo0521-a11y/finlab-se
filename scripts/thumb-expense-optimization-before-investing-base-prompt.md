【画像全体の雰囲気】
- ダークネイビーから漆黒のグラデーション背景（#050C23 → #0A1A4A）

- 目立つオブジェクトは絶対に置かない（後でテキストを重ねるため）

【カラーパレット】

- 1376×768px のPNG
- アスペクト比 16:9 横長
- テキスト・ロゴは絶対に入れないこと

【絶対にやらないでほしいこと】
- 白背景のインフォグラフィックにしない
- 4分割やテーブル形式のレイアウトにしない
- ✓や✗のチェックマークを入れない
