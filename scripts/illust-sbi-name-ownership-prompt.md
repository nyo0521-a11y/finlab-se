# 記事内イラスト生成プロンプト①：本人名義 vs 証券会社名義

記事: SBI証券のS株で高配当株100銘柄超を買った正直レビュー
配置先: 「良い点3. 単元未満でも自分名義の株主になれる」の章
保存先: `static/images/illust-sbi-name-ownership.png`

## 使い方
1. 新しいChatGPT（またはGemini）の会話を開始
2. 参考として既存の記事内イラスト（例: `scripts/dividend-vs-selling.png`）を添付し「このテイストで」と伝える
3. 以下のプロンプトを貼り付けて生成
4. 文字が崩れた場合は「日本語の文字をすべて正確にレンダリングし直して」と再指示
5. 完成画像をDownloadsに保存して、メインClaudeに「保存した」と伝える

---

```
日本語の投資ブログの記事内に埋め込む説明イラスト（1376×768px・16:9・横長）を作成してください。

【テーマ】
単元未満株（1株から買える株）の「名義」の違いを、左右対比で説明する図です。
SBI証券のS株は単元未満でも本人名義になるのに対し、SMBC日興証券のキンカブ（フロッギー）は単元未満分が証券会社名義になる、という違いを伝えます。

【全体のスタイル】
- 背景：白〜ごく薄いブルーグレー（#F5F7FB）。ダーク背景にしない
- 上部にタイトル、その下に左右2枚のカードを並べるレイアウト
- フォント：太い日本語ゴシック体。文字はすべてくっきりと
- フラットで清潔なインフォグラフィック調（写真調・3D調にしない）

【上部タイトル】
- 大きく太字で：「単元未満株の「名義」はどこが違う？」
- その下に小さめのグレー文字で：「同じ1株でも、株主名簿に載る名前が違う」

【左カード（緑系・メイン）】
- ヘッダー：緑色の帯に白文字で「SBI証券 S株」
- カード内の見出し：「1株でも あなたの名義」
- イラスト要素：株主名簿をかたどった台帳のアイコン。名簿の行に人物アイコンと「あなたの名前」というラベルが載っている様子
- 箇条書き（チェックマーク付き）：
  - 「株主名簿に自分の名前が載る」
  - 「配当金を株主として直接受け取る」
- カードの枠・アクセント色：グリーン（#22C55E系）

【右カード（オレンジ系・対比）】
- ヘッダー：オレンジ色の帯に白文字で「日興フロッギー（キンカブ）」
- カード内の見出し：「単元未満分は 証券会社の名義」
- イラスト要素：同じ台帳のアイコンだが、名簿の行にはビルのアイコンと「証券会社の名前」というラベルが載っており、人物アイコンはその外側にいる様子
- 箇条書き（「！」マーク付き）：
  - 「株主名簿に載るのは証券会社」
  - 「配当相当の金額は受け取れる」
- カードの枠・アクセント色：オレンジ〜アンバー（#F59E0B系）

【下部の注記】
小さめのグレー文字で1行：
「※どちらも配当（相当額）は受け取れます。違いは「株主名簿に誰の名前が載るか」です（執筆時点）」

【絶対にやらないでほしいこと】
- 日本語のテキストを英語に変換しない。上記の文言を一字一句そのまま使う
- 上記以外の文字・数字・ラベルを勝手に追加しない
- 実在の証券会社のロゴ・アプリ画面・カエルのキャラクターを描かない
- ダーク背景・写真調にしない
- 株価チャートやお金が飛び交うような過剰な装飾を入れない
```
