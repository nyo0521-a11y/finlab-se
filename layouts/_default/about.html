わたしのこと

新卒から金融系のシステム子会社に勤めています。これまでのキャリアは、銀行決済システム・クレジットカードのシステム・金融グループの国際規制に関するシステムと、お金そのものが動く領域を転々としてきました。SE（システムエンジニア）というと、多くの方が「プログラムを組む人」をイメージされるかと思いますが、わたしの仕事は少し毛色が違います。業務部門のユーザーが抱えているニーズを聞き取り、それをシステムにどう落とし込むかを設計する（あるいは、設計されたものがニーズに合っているかを確認する）という立場です。

なのでコードを書くプロというわけではないのですが、金融業務の中身と、その裏で動いている仕組みの全体像については、現場で揉まれながら自分なりの理解を積み上げてきたつもりです。このブログでは、その「仕組み側から見える景色」を、普段ニュースで金融に触れる側の方にも伝わる粒度で書いていきたいと思っています。

個人としては、日本株の高配当株を中心に長期で買い増していく投資スタイルです。短期売買は性格に合わないので、増配の継続性と財務の健全性を見ながら、配当でじわじわ家計を補強していくことを重視しています。

2026年からは Claude Code を本格的に使い始めて、家計・ポートフォリオ管理を自動化したり、自分が欲しいシミュレータをWebツールとして公開したりしています。「業務で触れている金融」「自分の口座で動いている運用」「AIで作る道具」、この3つが交わるところに、ちょうど書きたいテーマが集まる感覚があります。
