finlab-se は、金融SE（システムエンジニア）視点での資産形成・運用ブログです。
銀行・クレジットカードシステムの開発現場で得た知識と、自分の口座で検証した実体験をもとに、
以下の **4つのテーマ**を軸に発信しています。

## 1. 資産形成

家計改善・iDeCo・NISA・節税など、投資前の土台づくりに関する実践記録と考え方をまとめます。
高配当株投資の銘柄選定・ポートフォリオ管理も含まれます。

[→ 資産形成カテゴリの記事一覧](/categories/asset-building/)

## 2. AI活用

Claude Code を使った家計・資産管理の自動化、スプレッドシート連携、シミュレータ自作、
ブログ運営への活用事例をまとめます。

[→ AI活用カテゴリの記事一覧](/categories/ai/)

## 3. 金融の仕組み

銀行・決済システム・クレジットカード・全銀ネット・日銀ネット・加盟店手数料など、
お金が動く裏側を金融SE視点で分解します。

[→ 金融の仕組みカテゴリの記事一覧](/categories/finance-system/)

## 4. 金融ニュース

税制改正・新NISA・iDeCo・年収の壁など、生活と資産形成に効いてくる制度ニュースを
「いつ・誰に・いくら効くか」で深掘りします。

[→ 金融ニュースカテゴリの記事一覧](/categories/news/)
