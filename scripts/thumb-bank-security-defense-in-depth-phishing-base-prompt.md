## ステップ1：ベース画像生成プロンプト

```
日本語の個人投資ブログ用のサムネイル画像（1376×768px・16:9・横長）を作成してください。

【重要：これは「金融機関のセキュリティ・多重防御」をテーマにした記事のサムネイルです。仮想通貨やトレードの話とは無関係です。白背景のインフォグラフィックではなく、シネマティックなダークCG調の画像を作成してください。】

【画像全体の雰囲気】
- ダークネイビーから漆黒のグラデーション背景（#050C23 → #0A1A4A）
- 高級感のある金融テックの世界観
- 全体に薄い回路パターン／グリッドラインを敷く（深い青色で控えめに）
- 写真のようなリアル3D CG調
- テキスト・ロゴ・ラベルは一切入れないこと
- 後でテキストを左側に重ねるので、画面左45%は暗く保ち、目立つオブジェクトを置かないこと

【ビジュアル構成：多重防御のシールド】
画面右55%のエリアに、以下を配置：

▼ メインビジュアル：同心円状に配置された複数の発光シールド（多重防御を象徴）
- 中央：守られている対象として「金庫」または「金色の硬貨」が小さく光っている
- 中央のまわりに3〜5層の透明なエネルギーシールド（バリア）が同心円状に重なっている
- 各シールドは異なるグラデーション色：内側=濃いブルー、中間=エメラルドグリーン、外側=シアン
- シールドの表面に微かに六角形パターン・流れる光が走っている（テクノロジー感）

▼ 外側のシールドの外周に：
- 漆黒の中から赤い矢印やパーティクル（フィッシング攻撃の象徴）がシールドに弾かれている
- 矢印は青白くフラッシュして拡散している（攻撃が防御されているイメージ）
- ただし攻撃の量は控えめに（恐怖を煽らない）

▼ 全体に小さな粒子（緑・シアン・微かな赤）を漂わせる
▼ シールド全体に放射状のソフトグロー
▼ 画面右下に小さく「鍵マーク」のホログラム（暗号認証を象徴）

【画面左45%エリア】
- 暗いまま保ち、薄いグリッドラインだけが見える程度
- 目立つオブジェクトを置かない（後でテキストを重ねるため）

【カラーパレット】
- 背景：#050C23 → #0A1A4A
- 多重防御シールド：#3b82f6（青）→#4ADE80（緑）→#67E8F9（シアン）のグラデーション
- 攻撃の矢印・粒子：#dc2626 / #f97316（赤・オレンジ系）、控えめに
- 守られている対象（中央）：#F59E0B（アンバーゴールド）

【絶対にやらないでほしいこと】
- 白背景のインフォグラフィックにしない
- 攻撃を過度に強調しない（バリアが優位な印象を保つ）
- 4分割やテーブル形式にしない
- 人物・キャラクター・フード姿のハッカーは入れない
- 「危険」「警告」などの英語テキストを画像に入れない
- 写真のような銀行ビル・ATMを露骨に入れない（抽象的にとどめる）
```
