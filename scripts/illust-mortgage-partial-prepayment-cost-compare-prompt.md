# gpt-image-2プロンプト：挿絵「全額 vs 一部 費用対比」（実例の数字入り）

記事: 住宅ローンの金利上昇、一部繰り上げ返済にした理由と手数料（`content/drafts/mortgage-partial-prepayment.md`）
配置: H2「【実例】2012年・りそな銀行2,500万円で試算してみた」内、H3「残り約1年、100万円弱を残して一部繰り上げにした理由」の比較段落（「差し引き約15,000円の持ち出し」）の直後
保存先: `static/images/illust-mortgage-partial-prepayment-cost-compare.png`

## 設計意図

- 実例の核心「全額＝手数料22,000円（利息は0円に）／一部＝手数料0円＋残り利息 約7,000円 → 差額 約15,000円」を左右対比の1枚で見せる
- 一般論の章とは違い**筆者の実例の結論を示すパート**なので、「私はこちらを選んだ」と右側（一部）を明示してよい
- テイストはシリーズ統一（絵本調・暖色・同一人物）。ただし数字が主役なので、金額は大きくはっきり描かせる
- 数字は記事本文と完全一致させる: 22,000円／0円／約7,000円／約15,000円

住宅ローンの残りを「全額繰り上げ返済」した場合と「一部繰り上げ返済」した場合の、かかる費用の対比。筆者の実例では一部繰り上げ返済の方が約15,000円得だった、という比較を視覚的に伝えます。
