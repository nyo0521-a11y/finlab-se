# サムネイル ベース画像生成プロンプト

記事: SBI証券のS株で高配当株100銘柄超を買った正直レビュー
ファイル: `static/images/thumb-sbi-securities-honest-review-base.png`

## 使い方

1. **新しいChatGPT会話**を開始（過去の文脈を引きずらないため）
2. 既存サムネ1〜2枚（`thumb-crypto-exchange-financial-group-2026.png` 等）を添付し「このテイストで作って」と伝える
3. 以下のプロンプトをコピペして渡す
4. 生成されたベース画像をDownloadsに保存（テキスト合成はローカルのcompose.ps1で行うのでChatGPTでは不要）

---

```
日本語の個人投資ブログ用のサムネイル画像（1376×768px・16:9・横長）を作成してください。

【重要：これは「SBI証券の単元未満株（S株）で高配当株を100銘柄以上に少しずつ分散投資した実体験レビュー」に関する記事のサムネイルです。暗号資産・Bitcoinとは無関係です。白背景のインフォグラフィックではなく、シネマティックなダークCG調の画像を作成してください。】

【画像全体の雰囲気】
- ダークネイビーから漆黒のグラデーション背景（#050C23 → #0A1A4A）
- 高級感のある金融テックの世界観（BloombergやNikkeiの金融ビジュアルに近い）
- 全体に薄い回路パターン／グリッドラインを敷く（深い青色で控えめに）
- 写真のようなリアル3D CG調
- テキスト・ロゴ・ラベルは一切入れないこと
- 後でテキストを左側に重ねるので、画面左45%は暗く保ち、目立つオブジェクトを置かないこと

【ビジュアル構成】
画面右55%のエリアに、以下を配置：
- 中央：浮かび上がる立体的な株価チャート（緑色に発光する上昇ライン）を背景に、たくさんの小さな金色のコインが整然と積み上がっていく様子（少額をコツコツ積み上げるイメージ）
- コインの周囲：小さく発光する正方形のチップ・タイル群が格子状に広がる（多銘柄への分散をイメージ。100枚前後の小さなタイルが奥行きを持って並ぶ）
- アクセント色：エメラルドグリーン #4ADE80、アンバーゴールド #F59E0B
- 全体に小さな粒子（緑・金）を漂わせる
- 放射状のソフトグロー

【画面左45%エリア】
- 暗いまま保ち、薄いグリッドラインだけが見える程度
- 目立つオブジェクトを置かない（後でテキストを重ねるため）

【カラーパレット】
- 背景：#050C23 → #0A1A4A
- メイン：#4ADE80（エメラルドグリーン）／#F59E0B（アンバーゴールド）
- 粒子：緑と金の混色
- グリッドライン：深い青、非常に薄く

【サイズと形式】
- 1376×768px のPNG
- アスペクト比 16:9 横長
- テキスト・ロゴは絶対に入れないこと

【絶対にやらないでほしいこと】
- 白背景のインフォグラフィックにしない
- 4分割やテーブル形式のレイアウトにしない
- ✓や✗のチェックマークを入れない
- Bitcoinマーク・暗号資産の要素を入れない
- 証券会社のロゴ・実在のアプリ画面を入れない
```
