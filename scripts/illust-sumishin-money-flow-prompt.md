# 記事内イラスト生成プロンプト①：目的別口座のお金の流れ図（Gemini用）

記事: 住信SBIネット銀行の正直レビュー｜目的別口座の使い方と小遣い管理の実体験
配置先: 「一番気に入っている機能：目的別口座」の章（毎月のお金の流れの説明の後）
保存先: `static/images/illust-sumishin-money-flow.png`

## 使い方
1. 新しいGeminiの会話を開始
2. 参考として既存の記事内イラスト（例: `scripts/dividend-vs-selling.png`）を添付し「このテイストで」と伝える
3. 以下のプロンプトを貼り付けて生成
4. 文字が崩れたら「日本語の文字をすべて正確にレンダリングし直して」と再指示
5. 完成画像をDownloadsに保存して、メインClaudeに「保存した」と伝える

---

```
日本語の投資ブログの記事内に埋め込む説明図（1376×768px・16:9・横長）を作成してください。

【テーマ】
ネット銀行の「目的別口座」を使った小遣い管理のお金の流れ図です。
ポイントは、矢印の色で「自動化できる流れ（緑）」と「手動でしかできない流れ（オレンジ）」を区別することです。

【全体のスタイル】
- 背景：白〜ごく薄いブルーグレー（#F5F7FB）。ダーク背景にしない
- フラットで清潔なインフォグラフィック調（写真調・3D調にしない）
- フォント：太い日本語ゴシック体。文字はすべてくっきりと

【上部タイトル】
- 大きく太字で：「私の目的別口座の使い方とお金の流れ」
- その下に凡例を2つ並べる：
  - 緑の実線矢印のサンプル＋「自動化できる」
  - オレンジの破線矢印のサンプル＋「手動のみ」

【図の構成（5つの角丸ボックスと矢印）】

左上のボックス（グレー系）:
- 「家計の口座」

中央の大きめボックス（緑の枠で強調）:
- 「代表口座（決済用）」
- 小さい文字で「小遣いの入金・カード決済の引き落とし」

右側に縦に3つ並ぶボックス（青系）:
- 「目的別口座① つみたてNISA待機資金」
- 「目的別口座② パソコン購入資金」
- 「目的別口座③ 携帯購入資金」

左下のボックス（金色系）:
- 「SBIハイブリッド預金」
- 小さい文字で「配当金が入ってくる」

【矢印（向きと色を正確に）】
1. 「家計の口座」から「代表口座」へ：グレーの矢印、ラベル「小遣い」
2. 「代表口座」から右の目的別口座群へ：緑の実線矢印1本（途中で3つに分岐してもよい）、ラベル「定額自動振替で自動化できる」
3. 「目的別口座① つみたてNISA待機資金」から「代表口座」へ戻る：オレンジの破線矢印、ラベル「毎月のつみたて分を手動で移動」
4. 「SBIハイブリッド預金」から「目的別口座②③」へ：オレンジの破線矢印、ラベル「配当金を手動で積み立て」

【下部の注記】
小さめのグレー文字で1行：
「※自動化できるのは「代表口座から」の方向だけ（執筆時点）。取り崩し方向は手動になります」

【絶対にやらないでほしいこと】
- 日本語のテキストを英語に変換しない。上記の文言を一字一句そのまま使う
- 上記以外の文字・数字・ラベルを勝手に追加しない
- 実在の銀行・証券会社のロゴ・アプリ画面を描かない
- ダーク背景・写真調にしない
- 金額の数字を入れない
```
