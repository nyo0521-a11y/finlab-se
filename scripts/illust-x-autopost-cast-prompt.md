## 背景（なぜ作り直すか）
現行の cast 図は役割と流れが**間違っている**（Cloudflare Workersが投稿を実行・X APIがWorkersから依頼を受ける・GitHubが起点、と誤記）。
正しくは「Cloudflare Workers＝タイマー（起点）→ GitHub Actionsが投稿 → X API → X」。ブログ準備は前提として分離する。

4. 文字が崩れたら「日本語の文字をすべて正確にレンダリングし直して」と再指示
