# ②記事サムネ ベース画像生成プロンプト（ChatGPT / gpt-image-2）

記事: Xの自動投稿の"中身"をClaude Codeに任せる｜毎朝ニュースから紹介記事を選ぶ（claude-code-x-news-linked-post）
カテゴリ: AI活用
保存先（ベース）: `static/images/thumb-claude-code-x-news-linked-post-base.png`
完成サムネ: `static/images/x-news-post-thumbnail.png`（※文字はベース完成後にこちらでHTML合成します）
サイズ: 1376×768px（16:9）

## 進め方
1. **新しいChatGPT会話**を開く（過去の生成文脈を引きずらないため）
2. 既存サムネ `x-autopost-thumbnail.png` を1枚添付し「このダークCG調のテイストで作って」と伝える
3. 下記プロンプトを貼り付けて生成（**文字は一切入れない**ベース画像）
4. 生成画像をDownloadsに保存し、メインClaudeに「保存した」と伝える（→ 文字をHTMLで載せて完成・Draftアップします）

```
日本語の個人投資・AI活用ブログ用のサムネイル画像（1376×768px・16:9・横長）を作成してください。

【重要：これは「AI（Claude Code）が毎朝ニュースを読んで、その日にXへ投稿する記事を1本選ぶ」という"AIによる投稿内容の自動選定"に関する記事のサムネイルです。前回の「X自動投稿の仕組み」サムネ（Twitterの青い鳥・時計が主役）とは別物にしてください。Twitterの鳥を主役にしないこと。白背景のインフォグラフィックではなく、シネマティックなダークCG調にしてください。】

【画像全体の雰囲気】
- ダークネイビーから漆黒のグラデーション背景（#050C23 → #0A1A4A）
- 高級感のある金融テック／AIの世界観（BloombergやNikkeiの金融ビジュアルに近い）
- 全体に薄い回路パターン／グリッドラインを敷く（深い青色で控えめに）
- 写真のようなリアル3D CG調
- テキスト・ロゴ・ラベルは一切入れないこと
- 後でテキストを左側に重ねるので、画面左45%は暗く保ち、目立つオブジェクトを置かないこと

【ビジュアル構成】画面右55%のエリアに、以下を配置：
- 中央：AIの知性を象徴する、発光するニューラル／ブレイン状の球体コア（エメラルドグリーンに光る）
- その周囲：ホログラムのニュース記事カードや新聞の断片が複数、円を描くように浮遊・流れている
- AIコアが、その中の1枚のカードだけを緑色のスポットライトで選び出している（＝1本だけ選ぶ様子）
- 背景奥に、朝を示唆する淡い暖色（アンバー）のグロー（日の出のような光）を薄く
- 全体に小さな粒子（緑・金）を漂わせ、放射状のソフトグローをかける

【画面左45%エリア】
- 暗いまま保ち、薄いグリッドラインだけが見える程度
- 目立つオブジェクトを置かない（後でテキストを重ねるため）

【カラーパレット】
- 背景：#050C23 → #0A1A4A
- メイン：#4ADE80（エメラルドグリーン）／#F59E0B（アンバーゴールド）
- 粒子：緑と金の混色
- グリッドライン：深い青、非常に薄く

【サイズと形式】
- 1376×768px のPNG／アスペクト比 16:9 横長
- テキスト・ロゴは絶対に入れないこと

【絶対にやらないでほしいこと】
- Twitterの青い鳥を主役にしない（前回サムネとの差別化）
- 白背景のインフォグラフィックにしない
- 4分割やテーブル形式のレイアウトにしない
- ✓や✗のチェックマークを入れない
```
